LiveSync for WordPress
----------------------

LiveSync is a plugin for WordPress 1.5.x which allows a site administrator to 
easily synchronise a live WordPress database using a local copy (or indeed 
vice-versa), all within the comfort of the WordPress administration area.

Change Log
----------

Version 1.0 (30 November 2005)

* Initial Release

Installation Instructions
-------------------------

Download the file livesync.php and place this in a directory called /livesync
within your /<wordpress>/wp-content/plugins directory, then enable the plugin 
via the Plugins screen of WordPress admin.
